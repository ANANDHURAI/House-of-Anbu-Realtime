## 📌 Project Overview

A real-time chat and video calling application inspired by WhatsApp.
Includes instant messaging, presence tracking, read receipts, and peer-to-peer video calls using WebRTC.

---

## 🚀 Features

### 💬 Real-Time Chat

* WebSocket-based instant message delivery
* Read receipts (✓✓)
* Online/Offline presence
* Typing indicators
* Redis as real-time event broker

### 🎥 Video Calling (WebRTC)

* Peer-to-peer video calling
* Camera on/off
* Mute/unmute
* Call ringing popup + notifications
* Call connect/disconnect handling

### 🛠️ Backend (Django + DRF)

* Optimized REST APIs
* JWT Authentication
* PostgreSQL database (indexed chat storage)
* Redis for WebSocket event handling
* User, chat, messages, call logs API endpoints

### 🖥️ Frontend (React.js)

* Responsive WhatsApp-like UI
* Socket.io for live chat
* Integrated WebRTC UI
* Chat list + message bubbles
* Notification + call UI
